# CompareFK

группа в tg
https://t.me/CompareFK

## for support federal treasury

программа сверки 2-х множеств (выявление пересечения или разницы множеств)
1-е множество "master"
2-е множество "slave"

файл конфигурации -
```
mode - определяет режим работы "difference" - разница множеств / "intersection" - пересечение множеств
masterdsn и slavedsn - строка подключения к мастер источнику и слейв источнику
loglevel - уровень логирования - если нет проблем с работой программы и вы выполняете рутинные задачи то ставьте error
limit - кол-во элементов отправляемых как параметр в slave источник данных (отправлять 10-20 млн записей как параметр - не целесообразно - тк работать будет медленно)
ratelimit - кол-во одновременно работающих процессов на одном источнике данных (сколько скриптов работает параллельно
mastersql - путь к каталогу со скриптами к master источнику
slavesql - путь к скрипту для slave источника
```

## Windows
```
GOOS=windows GOARCH=amd64 go build -o ../../bin/compare.exe
```
